метод Пауэлла  v1.0
Теоретическое введение

Метод Пауэлла или метод сопряженных направлений находит минимум функции \(f(x)\) путем проведения последовательных одномерных поисков начиная с точки \(x_{0}^{(k)}\), вдоль системы полученных споряженных направлений. Нижние индексы обозначают векторы одного этапа, верхние номера этапов.

Поговорим о том, что такое споряженное направление. Предположим, что процесс минимизации \(f(x)\) начинается в точке \(x^{(0)}\), выбранным произвольно или по какому-либо определенному правилу. Для удобства возьмем его в виде единичного вектора, т.е \((s^{(0)})^Ts^{(0)}=1\). Тогда вектор \(x^{(1)}\) будет равен \(x^{(1)}=x^{(0)}+\lambda^{(0)} s^{(1)}\), где длина шага \(\lambda^{(0)}\) определяется из условия минимума \(f(x^{(0)}+\lambda s^{(0)})\) по \(\lambda\). После вычисления \(x^{(1)}\) надо выбрать новое направление. Это новое направлнние \(s^{(1)}\) называется сопряженным к старому направлению \(s^{(0)}\), если \((s^{(1)})[\bigtriangledown ^2f(x^{(0)})]s^{(0)}=0\).

Идея алгоритма Пауэлла в сущности заключается в том, что если на данном этапе поиска определяется минимум квадратичной функции \(f(x)\) вдоль каждого направления делается некоторый шаг, то полное перемещение от начала до \(p-го\) шага споряжено ко всем поднаправлениям поиска. Переход из точки \(x_{0}^{(k)}\) в точку \(x_{m}^{(k)}\) определяется формулой:

\[ x_{m}^{(k)}=x_{0}^{(k)}+ \sum_{i=0}^m \lambda_{i}^{(k)}s_{i}^{(k)}, i=1,...,m-1 \]

Алгоритмметода Пауэлла: Шаг 1: Начинать из \(x_{0}^{(k)}\). С помощью одномерного поиска определяем \(\lambda_{1}^{(k)}\) так, чтобы \(f(x_{0}^{(k)}+\lambda_1s_{1}^{(k)})\) принимала минимальное значение, \(x_{1}^{(k)} =x_{0}^{(k)}+\lambda_1^{(k)}s_{1}^{(k)}\) Начинать теперь из \(x_{1}^{(k)}\). С помощью одномерного поиска определяем \(\lambda_{2}^{(k)}\) так, \(f(x_{1}^{(k)}+\lambda_2s_{2}^{(k)})\) принимала минимальное значение, \(x_{2}^{(k)} =x_{1}^{(k)}+\lambda_2^{(k)}s_{2}^{(k)}\). Поиск проводится последовательно в каждом направлении, начиная с последней точки последовательности, пока не будут найдены все \(\lambda_{i}^{(k)}, i=1,...,n.\). \(\lambda_{i}^{(k)}\) используется на шаге 4. Шаг2: Поиск, осуществляемый в соответсвии с шагом 1, может привести к линейно-зависимым направлением поиска. Направление полного перемешений на \(k-ом\) этапе должно заменять предыдущее направление только в том случае, когда заменяющий вектор увеличивает определитель матрицы направлений поиска. Следовательно, после минимизации \(f(x)\) в каждом из \(n\) направлений, как на шаге 1, проводится один дополнительный шаг, величиной \((x_{n}^{(k)}-x_{0}^{(k)})\), соответсвующий полному перемещению на \(k-ом\) этапе и приводящий в точку \((x_{n+1}^{(k)}=2xn^{(k)}-x_{0}^{(k)})\). Шаг 3: Обозначим наибольшее значение \(f(x)\) в каком-либо направлении поиска на \(k-ом\) этапе через

\[ \bigtriangleup ^k=max[f(x_{i-1}^{(k)}-f(x_{i}^{(k)})] \]

Направление поиска, соответсвующее этому максимальному изменению \(f(x)\), обозначим через \(s_{m}^{(k)}\). Положим \(f_{1}=f(x_{0}^{(k)}),f_{2}=f(x_{n}^{(k)}),f_{3}=f(2x_{n}^{(k)}-x_{0}^{(k)})\), где \(x_{0}^{(k)}=(x_{0}^{(k)})\), \(x_{n}^{(k)}=x_{n-1}^{(k)}+\lambda_{n}^{(k)}s_{n}^{(k)}=x_{0}^{(k)} + \sum_{i=0}^n \lambda_{i}^{(k)}s_{i}^{(k)}\) Тогда если

\[ 1)f_3\geq f_1 или 2)(f_1-2f_2+f_3)+(f_1-f_2-\bigtriangleup^{(k)})^2\geq0.5\bigtriangleup^k(f_1-f_3)^2, \]

то использовать на \((k+1)-м\) этапе те же направления \(s_{1}^{(k)},...,s_{n}^{(k)}\), что и на \(k-ом\) \(s_{i}^{(k+1)}=s_{i}^{(k)}\), для \(i=1,...n,\) Сравнить \(f(x_{n}^{(k)})\) и \(f(x_{n+1}^{(k)})\), прировнять \(x_{0}^{(k+1)}\). шаг 4: Если не выполняется ни одно из условий

\[ 1)f_3\geq f_1 или 2)(f_1-2f_2+f_3)+(f_1-f_2-\bigtriangleup^{(k)})^2\geq0.5\bigtriangleup^k(f_1-f_3)^2, \]

то ищется минимум \(f(x)\) в направлении вектора \(s^{(k)}\), проведенного их \(x_{0}^{(k)}\) в \(x_{n}^{(k)}\), точка этого минимума берется как начальная для следущего этапа. \(s^{(k)}=x_{n}^{(k)}-x_{0}^{(k)}\) и заменяют \(s_{m}^{(k)}\) на \(s^{(k)}\) и перемещают в последний столбец. Шаг 5: Проверка критерия остановки:

\[ \frac{ |(x_{i}^{(k+1)}-x_{i}^{(k)})|}{|x_{i}^{(k)}+\epsilon|}<\epsilon \]
